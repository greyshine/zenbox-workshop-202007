### CSS Selectors
```css
body {...}                   -> Elementeselektor
.my-class {...}              -> Klassenselektor
#my-id {...}                 -> ID-Selektor
a[href] {...}                -> Attributselektor (minified)
input[type=email] {...}      -> Attributselektor
[data-my-attr="value"] {...} -> Attributselektor

section#data {}           <section id="data></section>
div.bar {}
```
